# URL Shortener

REST API service for creating short URLs and tracking redirect statistics.

## Stack

* PHP 8.3
* Laravel 13
* MySQL
* Docker

## Quick Start

### 1. Configure environment

```bash
make env
```

### 2. Start the project

```bash
make setup
```

The application will be available at:

```text
http://localhost:8076
```

## API Documentation

```text
http://localhost:8076/docs/api
```
